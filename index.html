Abschlussprojekt Basics

Erstelle ein Java-Programm mit den folgenden Grundelementen:

OOP Basics

  • Klassen erstellen - z.B. class Kunde { }
  • Attribute & Methoden - z.B. private String name; public void setName(String name) { }

Vererbung

  • Basisklasse & Kindklasse - z.B. class Fahrzeug { } class Auto extends Fahrzeug { }
  • (Bonus)Methoden überschreiben - z.B. @Override public String toString() { }

Exception Handling

  • try-catch Block - z.B. try { dateiLesen(); } catch (IOException e) { }
  • Eigene Exception - z.B. class UngültigeEingabeException extends Exception { }

Collections (eins von beiden)

  • Liste verwenden - z.B. List namen = new ArrayList<>();
  • Map verwenden - z.B. Map zuordnung = new HashMap<>();

Einfache Projektideen

  • Adressbuch: Kontakte speichern und verwalten
  • To-Do-Liste: Aufgaben erstellen und abhaken
  • Einfacher Shop: Produkte und Warenkorb verwalten
  • Inventarverwaltungssystem: Produkte verwalten, kategorisieren und Bestand führen
  • Bibliotheksverwaltung: Bücher ausleihen, zurückgeben und Nutzer verwalten
