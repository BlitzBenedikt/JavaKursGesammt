Was ist Java?

Java ist eine objektorientierte, klassenbasierte Programmiersprache, die 1995 von Sun Microsystems entwickelt wurde. Sie ist darauf ausgelegt, so wenige Implementierungsabhängigkeiten wie möglich zu haben.

Erste Schritte mit Java

Um mit Java zu programmieren, benötigen Sie das Java Development Kit (JDK).

Hello World Beispiel

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Grundlegende Konzepte

  • Variablen und Datentypen
  • Operatoren
  • Kontrollstrukturen (if, for, while)
  • Methoden

Übungsaufgaben

  1. Schreiben Sie ein Programm, das Ihren Namen ausgibt.
  2. Erstellen Sie ein Programm, das zwei Zahlen addiert und das Ergebnis ausgibt.
  3. Implementieren Sie ein Programm, das prüft, ob eine Zahl gerade oder ungerade ist.
